## What Is Verus?

**Verus** is a fundamentally different blockchain protocol focused on self-sovereign identity, decentralized finance, and privacy. It features VerusID (on-chain revocable/recoverable identities), user-launched currencies with protocol-level conversions, 50/50 hybrid mining/staking consensus, and full transaction privacy via zero-knowledge proofs â€” all at layer 1, with no smart contracts required.

Verus also enables anyone to launch independent, interoperable blockchains through **PBaaS (Public Blockchains as a Service)**, connected by trustless cross-chain bridges including an Ethereum bridge.
